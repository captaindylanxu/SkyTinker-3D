# 验证清单 / Verification Checklist

## ✅ 功能验证 / Feature Verification

### 1. 开发服务器 / Development Server
- [ ] 运行 `npm run dev`
- [ ] 访问 http://localhost:5174
- [ ] 页面正常加载，无控制台错误

### 2. 账号系统 / Account System

#### 首次访问 / First Visit
- [ ] 显示欢迎弹窗（AccountModal）
- [ ] 有三个按钮：创建新账号、找回账号、跳过

#### 创建新账号 / Create Account
- [ ] 点击"创建新账号"
- [ ] 输入昵称（例如：TestUser123）
- [ ] 输入4位PIN码（例如：1234）
- [ ] 点击"创建"
- [ ] 成功后弹窗关闭，进入教程

#### 昵称排重 / Nickname Uniqueness
- [ ] 清除浏览器数据
- [ ] 尝试创建相同昵称
- [ ] 显示错误："该昵称已被使用"

#### 找回账号 / Recover Account
- [ ] 清除浏览器数据
- [ ] 点击"找回账号"
- [ ] 输入已存在的昵称
- [ ] 输入正确的PIN码
- [ ] 成功恢复账号

#### PIN验证 / PIN Validation
- [ ] 输入正确昵称 + 错误PIN
- [ ] 显示错误："PIN 码错误"
- [ ] 输入不存在的昵称
- [ ] 显示错误："账号不存在"

#### 跳过功能 / Skip Function
- [ ] 点击"跳过"
- [ ] 直接进入教程（不创建账号）
